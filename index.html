ODEToolkit is a Java program that helps users calculate, visualize, and explore solutions to ordinary differential equations. It is made freely available for non-commercial download and use by Harvey Mudd College.

Currently in the seventh generation of development, the ODEToolkit project is a joint-effort of students and professors. This phase focuses on further improving the architecture of ODEToolkit, documenting the source code, as well as adding new features to the software.

ODEToolkit is developed in part with support from the National Science Foundation through DUE-0717490.
